Start
Step 1 - Read number n.
Step 2 - If n <= 1, Print "Not a prime number" else continue to step 3
Step 3 - For each number i from 2 to the square root of n:
         If  n % i == 0 Print  "Not a prime number".
          Else continue the loop.
Step 4 - If there are no divisors, Print "Prime number".
End
